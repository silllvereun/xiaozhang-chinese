# 샤오장의 중국어학습

빌드 도구 없는 단일 HTML 파일(`index.html`)로 만든 개인용 중국어 단어 암기 + 작문 연습 웹앱입니다.

## 실행 방법

- `index.html`을 브라우저로 바로 열거나, VSCode의 Live Server 확장으로 열면 됩니다.
- 별도 설치/빌드 과정이 없습니다.

## 기능

- 병음 또는 한자 입력 시험 (성조 없이 입력해도 정답 인정)
- 오답 횟수 + 미학습 기간 기반 가중 랜덤 출제
- 복습 모드 (틀린 단어만 모아 출제), 즐겨찾기 필터
- `Enter`로 확인/다음, `Tab`으로 패스
- 오답노트 / 패스노트
- 단어장 검색 · 추가 · 수정 · 삭제, 즐겨찾기 토글
- JSON 백업 내보내기 / 가져오기
- 브라우저 TTS로 발음 듣기 (설치된 중국어 음성을 자동으로 찾아서 사용)
- 문장 만들기 연습 — 규칙 기반 채점 (단어 포함 여부, 길이, 문장부호, 중국어 작성 여부 확인). 서버나 API 키 없이 완전히 클라이언트에서 동작합니다.
- 이메일 매직링크 로그인 — 로그인하면 학습 기록(오답/정답/패스/즐겨찾기)이 여러 기기에서 동기화됩니다.

## 데이터 저장

- **단어장**: Supabase(Postgres) `words` 테이블에 저장되는 공유 데이터입니다. 위키백과처럼 로그인 없이 누구나 조회·추가할 수 있고, 기존 단어 수정/삭제는 로그인한 사용자만 가능합니다.
- **개인 학습 기록**(오답/정답/패스 횟수, 즐겨찾기): 로그인 전에는 그 브라우저의 `localStorage`에만 저장됩니다. 이메일 매직링크로 로그인하면 그 순간부터 Supabase `word_progress` 테이블과 동기화되어, 다른 기기에서 같은 계정으로 로그인해도 기록이 이어집니다.
- Supabase 요청이 실패하면(오프라인 등) 마지막으로 불러온 단어 목록을 `localStorage` 캐시에서 사용합니다.

## 시드 데이터

`index.html` 안의 `RAW_WORDS` 객체에 카테고리별 단어가 `한자|병음|뜻` 형식의 텍스트 블록으로 들어 있으며, 최초 배포 시 이 391개(HSK1/2, HSKK초급, 회화, 업무, 금융, 거래소, IT/개발, 암호화폐 9개 그룹)를 Supabase `words` 테이블에 업로드해 두었습니다. 이후로는 앱이 `RAW_WORDS`가 아니라 Supabase에서 단어를 불러오며, 앱 내 `단어장 관리` 탭에서 자유롭게 추가/수정/삭제할 수 있습니다.
