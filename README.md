# 샤오장의 중국어학습

빌드 도구 없는 단일 HTML 파일(`index.html`)로 만든 개인용 중국어 단어 암기 + 작문 연습 웹앱입니다.

## 실행 방법

- `index.html`을 브라우저로 바로 열거나, VSCode의 Live Server 확장으로 열면 됩니다.
- 별도 설치/빌드 과정이 없습니다.

## 기능

- 병음 또는 한자 입력 시험 (성조 없이 입력해도 정답 인정)
- 오답 횟수 + 미학습 기간 기반 가중 랜덤 출제
- 복습 모드 (틀린 단어만 모아 출제), 즐겨찾기 필터
- `Enter`로 확인/다음, `Tab`으로 패스
- 오답노트 / 패스노트
- 단어장 검색 · 추가 · 수정 · 삭제, 즐겨찾기 토글
- JSON 백업 내보내기 / 가져오기
- 브라우저 TTS로 발음 듣기 (`zh-CN`)
- 문장 만들기 연습 — 규칙 기반 채점 (단어 포함 여부, 길이, 문장부호, 중국어 작성 여부 확인). 서버나 API 키 없이 완전히 클라이언트에서 동작합니다.

## 데이터 저장

현재는 브라우저 `localStorage`에 저장됩니다 (`wordList`, `wordStats` 키). 여러 기기 간 동기화가 필요 없다면 이대로 써도 되고, 필요하면 `단어장 관리` 탭의 백업 내보내기/가져오기로 기기 간 수동 이전이 가능합니다.

여러 기기·브라우저 동기화가 필요해지면 `index.html`의 `dbGet` / `dbSet` / `dbDelete` 함수 내부만 Supabase 등 실제 DB 클라이언트 호출로 교체하면 됩니다. 나머지 코드(`WORDS`, `wordStats`, CRUD, 렌더링)는 이 세 함수를 거쳐서만 저장소에 접근하므로 손댈 필요가 없습니다.

## 시드 데이터

`index.html` 안의 `RAW_WORDS` 객체에 카테고리별 단어가 `한자|병음|뜻` 형식의 텍스트 블록으로 들어 있습니다. 총 391개 단어(HSK1/2, HSKK초급, 회화, 업무, 금융, 거래소, IT/개발, 암호화폐 9개 그룹)로 시작하며, 앱 내 `단어장 관리` 탭에서 자유롭게 추가/수정/삭제할 수 있습니다.
